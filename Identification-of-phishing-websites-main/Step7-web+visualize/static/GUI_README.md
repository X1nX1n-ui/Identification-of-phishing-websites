## ⚠️ 注意事项

### 1. 模型文件位置
```
GUI_Modern.py 和 phishing_url_model.pkl 必须在同一目录！

你的目录/
├── GUI_Modern.py
└── phishing_url_model.pkl  ← 必须存在
```

### 2. 检测需要时间
```
- 特征提取: 1-5秒
- WHOIS查询: 可能较慢
- HTML获取: 取决于网站响应

总计: 通常 2-10秒
```

### 3. 网络要求
```
需要网络连接:
- DNS查询
- WHOIS查询
- HTML获取

离线无法检测！
```

## 🐛 常见问题

### Q: 窗口大小能调整吗？
A: 目前固定800x700，保证最佳显示效果。
   如需调整，修改代码第108行：
   ```python
   self.root.geometry("800x700")  # 改成你想要的尺寸
   ```

### Q: 能添加图标吗？
A: 可以！准备一个icon.ico文件，放在同目录即可。

### Q: 字体显示不正常？
A: 确保系统安装了 Microsoft YaHei UI 字体。
   如果没有，代码会自动使用默认字体。

### Q: 检测时窗口卡住？
A: 不会！使用了多线程，UI始终流畅。
