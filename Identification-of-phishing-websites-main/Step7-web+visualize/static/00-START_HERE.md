## 🆘 遇到问题？

### 常见问题

**Q: 双击HTML文件无反应？**
A: 在文件上右键 → 打开方式 → 选择浏览器

**Q: 图表显示不完整？**
A: 按 Ctrl+F5 清除缓存并刷新

**Q: Python提示找不到模块？**
A: 运行 `pip install flask flask-cors`

**Q: 端口5000被占用？**
A: 修改 server_fixed.py 中的端口号

### 获取帮助
1. 查看 QUICK_START.md 的故障排除部分
2. 查看 INSTALL.md 的详细说明
3. 运行 `python verify_system.py` 诊断问题
