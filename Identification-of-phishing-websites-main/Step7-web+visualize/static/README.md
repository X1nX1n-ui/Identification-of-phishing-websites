# 🛡️ 钓鱼网站检测系统 - 优化版 v2.0

> 基于机器学习的智能钓鱼网站检测系统，配备专业级数据可视化界面

## ✨ 核心特性

- 🎯 **高精度检测**: 准确率 91.5%，AUC 0.975
- 📊 **专业可视化**: 混淆矩阵、特征重要性、ROC曲线、训练历史
- 🎨 **优雅设计**: 莫兰迪大地色系，简约现代风格
- 📱 **响应式布局**: 完美适配桌面、平板、移动设备
- 🌐 **双语支持**: 中英文界面，专业术语双语标注

## 🚀 快速开始

### 方式1: 查看前端界面（推荐）
直接在浏览器中打开 `index.html` 即可查看完整的可视化界面。

### 方式2: 启动完整系统
```bash
# 1. 安装Python依赖
pip install flask flask-cors joblib numpy scikit-learn

# 2. 启动后端服务
python server_fixed.py

# 3. 访问系统
打开浏览器访问: http://localhost:5000
```

## 🔧 常见问题

### Q: 图表无法显示？
A: 确保在Web服务器环境下运行，检查浏览器控制台是否有错误。

### Q: 检测功能不工作？
A: 确保Python后端服务器正在运行，检查5000端口是否可用。

### Q: 图表显示不完整？
A: 清除浏览器缓存，刷新页面，或调整浏览器窗口大小。

## 🛠️ 开发指南

### 修改图表样式
编辑 `app.js` 中的 `colors` 对象:
```javascript
const colors = {
    primary: '#3A3632',
    warmOrange: '#D4886E',
    // ...
};
```

### 调整图表高度
编辑 `index.html` 中的CSS:
```css
#confusionMatrix {
    height: 600px;  /* 调整此值 */
}
```

### 更新模型数据
修改 `app.js` 中相应函数的数据:
```javascript
const data = [
    [6333, 621],
    [569, 6476]
];
```
